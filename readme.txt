== Description ==

IBNadds merupakan plugin untuk memasang Iklan IDBlogNetwork pada Blog Wordpress anda. Plugin ini hasil modifikasi dari Plugin AdSenseAnna (http://kopy-online.de/internet/google-adsense-in-the-wordpress-sidebar-widget/)

Semoga dengan adanya Plugin ini, bisa memberikan Kemudahan bagi Blogger untuk memasang Iklan IDBlogNetwork pada Blog anda

== Installation ==

1. Upload Folder IBN-Plugin ke folder wp-content/plugins/ 
2. Bisa juga upload IBN-Plugin.zip lewat menu Plugins - Add New - Upload
3. Aktifkan Plugin
4. Masukkan Blog ID Anda, (lihat Blog ID di http://idblognetwork.com/myblog.php)
5. Pilih Model Iklan YAng Sesuai Dengan Sidebar Anda
